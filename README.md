# Bắt đầu
  1. Mở cử sổ Terminal Ctrl + Shift + `
  2. npm install -> Enter
    - Để cài đặt các package cho phía Back-end
  3. cd client -> Enter -> npm install 
    - Để cài đặt các package cho phía Front-end
  4. cd .. -> Enter 
    - Để ra khỏi thư mục Client
  5. npm run dev 
    - Để chạy chương trình
